# Commands


## `-f`
### Usage:
-f [File Name to Create] -i ["Text input for the File"]
### Action:
Creates a File and write directly in to it


## `-d`
### Usage:
-d [Directory Name to Create]\
### Action:
Creates a Directory


## `-df`
### Usage:
-df [File Name to Delete]\
Action:\
Deletes a File


## `-dd`
### Usage:
-dd [Directory Name to Delete]\
### Action:
Deletes a Directory


## `-u`
### Action:
Updates Cy


## `-v`
### Action:
Shows the Version of Cy
